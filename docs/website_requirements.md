## PROJECT OVERVIEW

- Client: Dr. Parul Jangra, Dental Theatre — multi-location dental practice in Panipat.
- Scope: Design and development of a new website covering all clinic locations.
- Priority: Strong design with animations, patient-friendly UX
- Reference style: Modern, professional dental/healthcare websites with rich visuals.

---

## TECH STACK & PLATFORM

- Platform: To be decided by dev team — must support animations
- Mobile-first responsive design required across all pages.
- Google Business Profile setup for each clinic location (separate profiles, linked to website).
- SEO-friendly architecture — clean URLs, meta tags, schema markup for local search, json-ld etc.

---

## REQUIRED PAGES

| Page | Description |
|---|---|
| Home Page | Hero section, tagline, CTA to book appointment, quick links to services and gallery. |
| About / Doctor Profile | Dr. Parul's experience, qualifications, credentials, years in practice, personal story. |
| Services | List of all dental treatments offered. Each service with brief description. SEO-optimised. |
| Gallery | Photo gallery section — raw working photos of Dr. Parul at work, clinic interiors, procedures. |
| Testimonials | Patient reviews and testimonials. Can be static or integrated with Google Reviews. |
| Book Appointment | WhatsApp-based booking flow-> click-to-WhatsApp button. |
| Contact / Locations | All clinic locations with addresses, maps, phone numbers, and hours. |

## FUNCTIONAL REQUIREMENTS

### FR-01 WhatsApp Appointment Booking
Book Appointment button/CTA on homepage and service pages opens WhatsApp chat with pre-filled message. Must work on both mobile and desktop.
This will be a floating wp icon on the bottom right corner of the screen.
**Must Have**

### FR-03 SEO — Local Search Optimisation
All pages must be SEO optimised. Local SEO for Panipat. Schema markup for dental practice, doctor profile, and each location. Google Business Profile integration for all three clinic locations.
**Must Have**

### FR-04 Image Gallery System
Dedicated gallery page with filterable photo categories (clinic, doctor at work, procedures, patient setups). Lightbox view on click. Optimised for fast loading.
**Must Have**

### FR-05 Testimonials Section
Testimonials displayed on homepage and a dedicated section. Option to pull from Google Reviews via embed or display manually uploaded testimonials.
**Must Have**

### FR-06 Social Media Links
All social media profiles (Instagram, Facebook, YouTube, etc.) linked from header/footer and from the About page.
**Must Have**

### FR-07 Multi-Location Support
Website must display all clinic locations clearly. Each location with address, map embed, phone number, and hours. Separate Google Business Profiles set up for each.
**Must Have**

### FR-08 Animations & Motion Design
Client specifically requested animation — entrance animations on scroll, smooth transitions between sections, animated hero section. Should feel modern and premium, not template-like.
**Must Have**

### FR-09 Mobile Responsive
Full mobile responsiveness across all pages and features. WhatsApp booking and gallery must work correctly on mobile.
**Must Have**

### FR-10 Doctor Experience & Credentials Profile
Dedicated section or page showing years of experience, qualifications, institutions, certifications, and any awards or recognitions.
**Should Have**

### FR-11 Contact Form
Standard contact form with name, phone, message. Submissions sent to clinic email or WhatsApp.
**Should Have** // will see this feature in future.

### FR-12 Google Maps Embed
Embedded Google Maps for each clinic location on the Contact/Locations page.
**Should Have**

## NON-FUNCTIONAL REQUIREMENTS

- Page load speed: Under 3 seconds on mobile (4G). Images to be compressed and lazy-loaded.
- SEO: All pages must have unique meta title, meta description, and OG tags.
- Accessibility: Basic WCAG compliance — alt tags on all images, readable font sizes.
- Browser support: Chrome, Safari, Firefox, Edge — latest two versions.
- SSL certificate: HTTPS required.
- Analytics: Google Analytics 4 and Google Search Console must be integrated.

## DELIVERABLES

- Fully functional, deployed website with all pages listed above.
- Google Business Profile setup for all three clinic locations.
- GA4 and Search Console linked and verified.
- All pages SEO-ready with meta tags and local schema markup.
